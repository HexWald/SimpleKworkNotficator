## Шаги по установке

1. **Скачайте Python**
   - Перейдите на [официальный сайт Python](https://www.python.org/downloads/).
   - Выберите версию Python, которая соответствует вашей операционной системе (Windows, macOS, Linux).

2. **Установите Python**
   - Запустите загруженный установщик.
   - Важно: установите галочку на опции **"Add Python to PATH"**, если такая есть.
   - Следуйте инструкциям установщика, чтобы завершить установку.

3. **Проверьте установку**
   - Откройте терминал (или командную строку) и введите следующую команду:
     ```bash
     python --version
     ```
   - Если установка прошла успешно, вы увидите установленную версию Python.
